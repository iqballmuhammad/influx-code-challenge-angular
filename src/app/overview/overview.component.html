Influx Code Challenge

There are over 3000 known planets in the universe! We need an app to help us find planets quickly.

The API

The planet database can be accessed via the following API endpoint:

https://influx.com/code-challenge/planets.json

The Challenge

Using the JavaScript framework of your choice:

Build a single page application to fetch data from the API and present it to the user.

The user should be able to browse all of the planets in the Star Wars universe.
